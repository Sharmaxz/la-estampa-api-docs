Vi Unge¶

Project develop in Python 3.5, Django 1.11.5 and Django Rest Framework 3.6.3 to provide REST API for Vi Unge magazine App.

Requirements¶

  • Python (3.4, 3.5, 3.6)

  • Django (1.10, 1.11)

Instalation¶

We strongly recommended to use virtualenv, but if you are not familiar with virtualenv, you can read more in ‘’https://virtualenv.pypa.io/en/latest/’’

$ pip install -r requirements.txt

Environment Variables¶

We have to create a .env file in project root folder with this informations. If you don’t have access to theses keys contact the development team.

DEBUG=''
SECRET_KEY=''
DATABASE_URL=''
AWS_ACCESS_KEY_ID=''
AWS_STORAGE_BUCKET_NAME=''
AWS_SECRET_ACCESS_KEY=''
STATIC_URL=/static/
MEDIA_URL=/media/

Run¶

Run the following command:

$ python manage.py runserver

EndPoints¶

Our architecture follow Representational State Transfer (REST) architectural style for for distributed hypermedia systems (API’s).

  • Authenticating Requests
  • Category
  • Video Post
  • Report
  • Generate Presigned Post for AWS S3 Bucket
  • Social Login Endpoints
    • Convert Token
    • Invalidate Sessions
